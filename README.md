# COBOL Banking Programs

This repository contains simple COBOL programs based on basic banking and validation scenarios.  
These programs are useful for beginners to understand core COBOL concepts such as input/output, conditional statements, and arithmetic operations.

---

## Programs Included

### 1. Age Eligibility Check
- Accepts age from the user
- Checks whether the customer is eligible (age > 18)
- Displays eligibility status

---

### 2. Minimum Balance Validation
- Accepts account balance
- Compares it with a predefined minimum balance (₹1000)
- Displays whether minimum balance is maintained

---

### 3. PIN Verification
- Stores a predefined PIN
- Accepts PIN from the user
- Grants or denies access based on correctness

---

### 4. Low Balance Warning
- Demonstrates balance-related validation logic
- Can be extended to show warnings when balance is low

---

### 5. Deposit Transaction
- Accepts current balance
- Accepts deposit amount
- Validates deposit amount (> 0)
- Adds deposit amount to balance
- Displays updated balance

---

## Technologies Used
- **Language:** COBOL
- **Concepts Covered:**
  - ACCEPT / DISPLAY
  - IF–ELSE conditions
  - Arithmetic operations
  - Working-Storage variables

---

## How to Run
1. Open the program in a COBOL-supported environment (Mainframe, GnuCOBOL, etc.)
2. Compile the program
3. Execute and provide input as prompted

---

## Purpose
These programs were created as part of learning and practice for COBOL basics and banking-related logic.  
They are suitable for academic work, assessments, and interview preparation.
