Q1.Age Eligibility Check

IDENTIFICATION DIVISION.
PROGRAM-ID.AGE.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 WS-AGE PIC X(2).
PROCEDURE DIVISION.
DISPLAY 'ENTER THE AGE: '.
ACCEPT WS-AGE.
IF WS-AGE > 18
DISPLAY 'Customer is eligible'
ELSE 
DISPLAY 'Customer is not eligible'
END-IF.
STOP RUN.

Q2.Minimum Balance Validation

IDENTIFICATION DIVISION.
PROGRAM-ID.BALANCE.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 MINIMUMBALANCE PIC 9(4) VALUE 1000.
01 BALANCE PIC 9(4).
PROCEDURE DIVISION.
DISPLAY ' enter the balance: '.
ACCEPT BALANCE.
IF BALANCE >= MINIMUMBALANCE
DISPLAY 'Minimum balance maintained'
ELSE 
DISPLAY 'Minimum balance not maintained'
END-IF.
STOP RUN.

5.Deposit Transaction

IDENTIFICATION DIVISION.
PROGRAM-ID.DEPOSIT.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 CURRENT-BALANCE PIC 9(4).
01 DEPOSIT-AMOUNT PIC 9(4).
01 UPDATED-BALANCE PIC 9(4).
PROCEDURE DIVISION.
DISPLAY 'ENTER THE CURRENT BALANCE'.
ACCEPT CURRENT-BALANCE.
DISPLAY 'ENTER THE DEPOSIT AMOUNT'.
ACCEPT DEPOSIT-AMOUNT.
IF DEPOSIT-AMOUNT > 0
  COMPUTE UPDATED-BALANCE= DEPOSIT-AMOUNT + CURRENT-BALANCE
ELSE
  DISPLAY'ENTER THE VALID AMOUNT'
END-IF.
DISPLAY UPDATED-BALANCE.
STOP RUN.
